# Token Info

**API Key Required:** No

Search for any token and retrieve metadata, live market data, and candlestick charts.

## Capabilities

- **Search** — find tokens by name, symbol, or contract address
- **Detail** — full metadata (name, symbol, supply, holders, website, socials)
- **Live Price** — real-time market data
- **K-line Charts** — candlestick data at 1m, 5m, 1h, 1d intervals
